PSR+ 是传统 Windows 步骤记录器的演变

下载 PSR+

文件生成

它将所有这些内容打包成一个 html 文件。这个 .html 文件是一个网页，可以在任何兼容的浏览器或文本编辑器中打开。

截屏

每次点击鼠标或按下相关按键时，都会自动截取屏幕截图。它会聚焦于操作发生的区域。

自动描述

基于屏幕截图和事件检测，它会自动生成事件发生的文本描述。例如：“在‘Document.docx - Word’窗口中左键单击‘保存按钮’”。

三个基本步骤轻松搞定

无需复杂的设置、选项繁多的菜单，也无需理解专业术语。只需即插即用，轻松共享。PSR+ 旨在节省您的时间，让您轻松便捷地完成文档编写和问题解释。

  • 开始并记录。
  • 添加评论（如果需要）。
  • 停下来并分享。

为您设计的功能：

这些功能使 PSR+ 成为任何需要清晰高效地记录数字交互的人的强大工具。

智能细致的录音

情境截图：它不会录制连续的视频，而是在关键时刻（例如，每次互动后）截取高质量的截图。这样可以保持文件轻量，并专注于重要内容。

改进的注释和评论工具

自定义注释：允许用户在录音的任何时间点添加解释性文字，以说明操作背景、描述预期结果或指出偏差。

优化的可用性和界面

直观的界面：比原始 PSR 更清晰、更易于使用的设计，即使对于技术水平较低的用户，也能简化记录和审查步骤。

高效的报告生成和共享

交互式报告：将所有数据（屏幕截图、描述、评论）编译成 HTML 报告，可在任何浏览器中轻松查看。

直观的视觉亮点

用于标记或突出显示屏幕截图中特定区域的功能。使用箭头引导用户注意重要按钮、文本字段、错误消息等。

准备立即共享

保存后，包含报告的 HTML 文件可以立即发送给支持团队、同事、客户或学生，确保有效、无障碍的视觉沟通。
